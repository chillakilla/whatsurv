### 1. 프로젝트 명: WhatSurv
### 2. 프로젝트 소개 : WhatSurv? 은 IT 종사자 및 관계자가 사용자의 피드백을 간편하고, 빠르게 수집할 수 있도록 돕는 설문조사 웹사이트로 기획했습니다.
### 3.  프로젝트 기간 : 24.01.04 - 24.02.08 

### 4. 함께 한 팀원 :
리더 김동학, 부리더 이다원, 이희원, 최광희

### 5. 주요 목표:
- 로그인(구글 포함)과 회원가입 
- IT 관련 설문조사 CRUD
- 가벼운 설문조사 CRUD
